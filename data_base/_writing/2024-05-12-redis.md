# \[DB] Redis

>Remote Dictionary Server이다. 

- 비정형 데이터를 처리하기 위한 오픈 소스 기반의 NoSQL이다.
	- Key-Value 쌍으로 이루어진 구조
- In-Memory 데이터 구조를 가졌다.
	- RAM에 데이터를 올려 처리하는 방법이다.
	- 휘발성을 가진다.
- 사용자의 필요에 따라 여러 형태로 사용할 수 있다!

# \[AWS] RDS

>AWS 서버에 데이터를 정형화해 저장해야 하는 경우가 있다.\
>
>이런 경우 Amazon EBS 대신 Amazon RDS를 사용하면 더 좋다!

- Amazon RDS(Relational Database Service)는 AWS 클라우드에서 관계형 데이터베이스를 더 쉽게 설치, 운영 및 확장할 수 있는 웹 서비스이다.
	- 엑셀과 비슷하다고 느껴진다.

# \[AWS] OpenSearch

- AWS에서 제공하는 OpenSearch는 오픈 소스이다.
	- 따라서 AWS에서만 사용 가능한 것이 아니라 직접 구축해서 사용할 수 있다!
- 검색 엔진 기능 및 Vector Search 기능도 제공한다.
	- 검색 엔진은 Elastic Search를 활용한다.
